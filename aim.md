# 摸鱼小助手 - 项目目标规划

## 一、项目概述

一款跨平台桌面摸鱼小工具，帮助用户在上班/学习间隙放松身心，同时避免被发现。

## 二、技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Tauri 2.x |
| 前端 | Vue 3 + TypeScript + Vite |
| 状态管理 | Pinia |
| 路由 | Vue Router 5 |
| 后端 | Rust |
| 样式 | CSS Variables (支持暗黑模式) |

## 三、核心功能

### 1. 基础功能
- [x] 自定义窗口标题（伪装标题）
- [x] 窗口透明度调节
- [x] 老板键（Ctrl+~ 快速隐藏）
- [x] 系统托盘（隐藏到托盘）
- [x] 暗黑模式切换
- [x] 侧边栏折叠与状态缓存

### 2. 小说阅读
- [x] 扫描本地 txt 文件
- [x] 阅读器界面（翻页、进度记忆）
- [x] 广告过滤

### 3. 音乐播放
- [x] 扫描本地音乐文件
- [x] 播放/暂停/上一首/下一首
- [x] 进度条拖动
- [x] 音量控制

### 4. 热点新闻
- [x] 知乎热榜数据
- [x] 标题+热度展示
- [x] 外部链接打开

### 5. 实用工具
- [x] 倒计时器（番茄钟）
- [x] 剪贴板监控（敏感词过滤）
